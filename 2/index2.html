5 Написать функцию калькулятор (calc). Детали задания:
calc может выполнять, только арифметические операции +-/*;
calc может считать более одной арифметической операции (1+2+...+5-4*...*n+m);
Проверка валидности условия, если выражение не валидно, возвращать false
Минимальное количество if/else (в идеале без if)

Решение:
function calc(str) {
return /^(?:\d+([*+-]|\/(?!0)))+\d+$/.test(str) ? eval(str): false;
}
