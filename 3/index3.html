Выражения и операции
Упражнения

1)В два раза старше​
Ваша функция принимает два аргумента:
текущий возраст отца (лет)
текущий возраст сына (лет)
Подсчитайте, сколько лет назад отец был вдвое старше сына (или через сколько лет он будет вдвое старше).

2)Считайте нечетные числа ниже n
Если задано число n, вернуть количество положительных нечетных чисел меньше n
oddCount(7) //=> 3, i.e [1, 3, 5]
oddCount(15) //=> 7, i.e [1, 3, 5, 7, 9, 11, 13]


3)Преобразуйте строку в число!
Нам нужна функция, которая может преобразовывать строку в число. Какие способы добиться этого вы знаете?
Примечание: не волнуйтесь, все входные данные будут строками, и каждая строка является вполне допустимым представлением целого числа.

stringToNumber('1234') == 1234
stringToNumber('605') == 605
stringToNumber('1405') == 1405
stringToNumber('-7') == -7

4)Найдите числа, которые делятся на заданное число​
Завершите функцию, которая принимает два аргумента и возвращает все числа, которые делятся на данный делитель. Первый аргумент - это массив чисел, а второй - делитель. Пример​
divisibleBy([1, 2, 3, 4, 5, 6], 2) == [2, 4, 6]
